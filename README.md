# Quickapply – After Effects Script

Quickapply is a workflow utility for Adobe After Effects designed to speed up text animation and preset workflows.

## Features
• Batch apply animation presets (.ffx)
• Apply presets at layer In Point or comp start
• Option to target only text layers
• UnPrecompose text layers from precomps
• Attempts to preserve timing and transforms

## Installation
1. Download Quickapply.jsxbin
2. Place inside:

After Effects > Scripts > ScriptUI Panels

3. Restart After Effects
4. Open via:

Window → Quickapply
